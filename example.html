2 Projects for writing classes

2.1 Complex numbers

In mathematics, a complex number is an element of a number system that extends the real numbers with specific element denoted \(\mathrm{i}\), called the imaginary unit and satisfying the equation \(\mathrm{i}^2 = -1\) (or equivalently \(\mathrm{i} = \sqrt{-1}\)). Every complex number \(z\) can be expressed in the form \(z = a + b \mathrm{i}\), where \(a \in \mathbb{R}\) and \(b \in \mathbb{R}\) are real numbers. Two complex numbers \(z_1 = a_1 + b_1 \mathrm{i}\) and \(z_2 = a_2 + b_2 \mathrm{i}\) are added according to the rule \[ z_1 + z_2 = (a_1 + a_2) + (b_1 + b_2) \mathrm{i}, \] and multiplied by each other according to the rule \[ z_1 \cdot z_2 = (a_1 \cdot a_2 - b_1 \cdot b_2) + (a_1 \cdot b_2 + a_2 \cdot b_1) \mathrm{i}. \]

Your task is to write C++ code to define a class ComplexNumber, its data members, member functions, and non-member functions (if there are) to reflect the above description. You should see (a) – (h) questions.

  1. Write code to define the class ComplexNumber with two private data members, named re and im.

  2. Write a constructor receiving two input arguments re and im with default value \(0\) to initialize the data members described in (a). Use member initializer list (not the assignment statements) to initialize these data members.

  3. Write a function overloading the operator << of the object std::ostream so that the statement std::cout << comp_num; works for the ComplexNumber object comp_num. You must decide the print-out format that makes sense to the user. For example, consider the following numbers: \(1 + 0\,\mathrm{i} = 1\) and \(0 + 2\,\mathrm{i} = 2\,\mathrm{i}\).

  4. Write functions to overload the operator + (plus sign) and the operator * (multiplication sign) to reflect the addition and multiplication of two complex numbers, respectively. Essentially, the following statements should work after overloading the operators <<, + and * for the complex numbers:

    std::cout << a + b;
std::cout << a * b;

  5. Write a member function with name pow to compute the power \(z^{n}\) of the complex number \(z\) with \(n\) being a non-negative integer. Recall that \[ z^{0} = 1,\quad z^{1} = z, \quad z^{2} = z \cdot z, \quad z^{3} = z\cdot z \cdot z, \quad z^{n} = \underbrace{z \cdot \cdots \cdot z}_{n \text{ terms}}. \]

  6. Write a function to overload the assignment operator = for a ComplexNumber object so that the following statement works:

    c = a + b;  // a, b, c are all ComplexNumber objects

  7. Write a class RealNumber derived from the class ComplexNumber you have defined above. The class RealNumber must have a constructor receiving one double number and make sure that the imaginary part derived from the parent class is equal to zero.

  8. Write a main function to perform the following tasks:

  • Create an array of five ComplexNumber elements using dynamic memory allocation with a pointer to ComplexNumber. Remember to clean up the memory when you do not need the array anymore.
  • Initialize these five complex numbers using the for-loop to obtain five numbers (note the pattern of the real parts and imaginary parts of these numbers): \[ z_{1} = 1 + 2\,\mathrm{i},\quad z_{2} = 2 + 3\,\mathrm{i}, \quad z_3 = 3 + 4\,\mathrm{i}, \quad z_4 = 4 + 5\,\mathrm{i}, \quad z_5 = 5 + 6\,\mathrm{i}. \]
  1. Compute the sum \(\displaystyle s = \sum\limits_{j=1}^{5} z_{j}\) of these five numbers using the for-loop and print the resulted sum \(s\) to the console.
  2. With \(z = 1 + 2\,\mathrm{i}\), compute the expression \[ \Lambda = 1 + z + z^2 + z^3 + z^4 + z^5 \] using the for-loop.

Note that you can write any helper functions that are not defined by answering the above questions to complete the main function and other functions.

Hint  static_cast<double>(12) will convert the int \(12\) into the double \(12.0\).

2.2 Fractions

Your task is to write C++ code to define a class Fraction, its data members, member functions, and non-member functions (if applicable) to fulfill the following description.

  1. An object of type Fraction has two private data members of integer type, named nom and denom.

  2. Write a constructor receiving two input arguments nom and denom with appropriate default values to initialize the data members. Use a member initializer list (not assignment statements). Ensure that the denominator cannot be zero: provide a warning and set the denominator to \(1\) if \(0\) is provided.

  3. Write a member function to perform the addition of two fractions. You need to decide what the inputs and outputs of this function should be.

  4. Write a member function to perform the multiplication of two fractions.

  5. Overload the operator << for std::ostream so that std::cout << f; works for a Fraction object. The function must handle the following formatting:

    • If the numerator is \(0\), print 0.
    • If the denominator is \(1\), print only the numerator (e.g., \(42/1\) prints as 42).
    • If \(f > 0\), print the fraction in the form a/b (e.g., \(2/3\) for both \(2/3\) and \(-2/-3\)).
    • If \(f < 0\), print the negative sign in front of the positive fraction (e.g., \(2/-3\) prints as -2/3).

  6. Overload the operator + (plus) and operator * (multiplication) so that the following statements work for two Fraction objects a and b:

    std::cout << a + b; // Equivalent to std::cout << a.add(b)
std::cout << a * b; // Equivalent to std::cout << a.multiply_by(b)

  7. Overload the assignment operator = for a Fraction object so that the following works:

    c = a + b;  // a, b, and c are all Fraction objects

  8. Write a member function reduce that reduces a fraction to its irreducible form using std::__gcd(int, int) from #include <bits/stdc++.h>. For example, the GCD of \(100\) and \(40\) is \(20\), so the fraction should be reduced accordingly.

  9. Finally, write a main function to perform the following:

    • Create a std::vector containing five Fraction objects with the values: \[f_1 = \frac{1}{2}, \quad f_2 = \frac{1}{3}, \quad f_3 = \frac{1}{4}, \quad f_4 = \frac{1}{5}, \quad f_5 = \frac{1}{6}\] using a for-loop and push_back().
    • Compute the product using a for-loop: \[p = \prod_{i=1}^{5} f_{i} = f_1 \cdot f_2 \cdot f_3 \cdot f_4 \cdot f_5 = \frac{1}{2 \cdot 3 \cdot 4 \cdot 5 \cdot 6}\]
    • Print the resulting product to the console.
